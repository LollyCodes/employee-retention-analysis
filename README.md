# Employee Retention Analysis

## Project Overview
This project analyzes employee data to identify factors that influence employee retention. The goal is to determine which variables most strongly affect whether employees stay or leave a company.

## Dataset
The dataset contains employee information such as:
- Satisfaction level
- Last evaluation score
- Number of projects
- Monthly working hours
- Time spent at company
- Work accidents
- Promotion history
- Department
- Salary
- Attrition status (left or stayed)

## Exploratory Data Analysis
Exploratory analysis was performed to:
- Understand data distribution
- Identify correlations
- Detect important features influencing retention

## Key Findings
- Satisfaction level is the strongest predictor of employee retention.
- Salary level affects employee turnover.
- Department also influences retention patterns.
- Work accidents and tenure show moderate impact.

## Model Used
A Logistic Regression model was built using selected features.

## Model Accuracy
Accuracy achieved: **75.4%**

## Conclusion
The analysis demonstrates that machine learning can effectively predict employee turnover and help organizations make informed HR decisions.
